Conectá tu repositorio

Los datos viven en el repo privado . Necesitás un token personal de GitHub (se guarda solo en este navegador).

  1. Entrá a github.com/settings/personal-access-tokens → Generate new token (fine-grained).
  2. En Repository access elegí Only select repositories → .
  3. En Permissions → Repository → Contents poné Read and write.
  4. Generá el token (empieza con github_pat_) y pegalo acá abajo.
